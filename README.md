# Gofin - API para aplicação de fiananças pessoais

v1 Features:
- [ ] Cadastro
    - Ao realizar o cadastro, um cofre deve ser criado.
- [ ] Login
- [ ] CRUD gastos
    - O usuario pode cadastrar incomes/outcomes para um cofre.

v2 Features:
- [ ] Recuperar senha
- [ ] Email de boas vindas
- [ ] Criar outros cofres
    - Geral
    - Cota
    - Recorrente
